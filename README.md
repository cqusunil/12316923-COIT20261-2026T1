## 📖 About This Repository

This repository contains all **weekly tutorial submissions** for the unit **COIT20261 – Network Routing and Switching** at CQUniversity Australia. Each week includes a detailed markdown write-up of completed practical tasks, supported by screenshots taken during the lab exercises.

The tutorials use **GNS3** (Graphical Network Simulator-3) running inside an **Oracle VirtualBox** virtual machine, accessed through a browser-based Web UI. No additional GNS3 software installation is required on the local machine.
